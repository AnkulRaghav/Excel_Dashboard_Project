# Excel_Dashboard_Project
📊 Excel Data Analytics Project | Sales Dashboard
I’m excited to share my latest Data Analytics project built in Microsoft Excel 🚀
🔍 Project Overview:
Analyzed a sales dataset to extract meaningful insights using Excel tools like Pivot Tables, Charts, and Slicers.
📌 Key Highlights:
✔️ Cleaned and transformed raw data (handled missing values, formatted dates, removed duplicates)
✔️ Created interactive Sales Dashboard
✔️ Used Pivot Tables for multi-level analysis (State, Category, Payment Mode)
✔️ Designed KPIs:
• Total Quantity: 12,745
• Total Amount: ₹61,82,639
• Total Profit: ₹16,40,697
📊 Insights Generated:
🔹 Furniture category generated the highest sales
🔹 Ohio & California are top-performing states
🔹 Credit Card & UPI are dominant payment modes
🔹 Clear yearly trend helps in performance tracking
🛠 Tools Used:
Microsoft Excel | Pivot Tables | Data Cleaning | Data Visualization
